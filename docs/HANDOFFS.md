# Handoff File Convention

Use this naming rule for future handoff documents in this repo:

- Directory: `docs/`
- Filename: `YYYY-MM-DD-<short-topic>-handoff.md`

Examples:

- `2026-03-14-setup-and-troubleshooting-handoff.md`
- `2026-03-20-auth-debug-handoff.md`
- `2026-03-22-db-recovery-handoff.md`

Guidelines:

- Create a new dated file for a new session or milestone.
- Do not overwrite older handoffs unless the user explicitly asks for a rewrite.
- Keep the topic short and specific.
- To find the newest handoff, sort by filename or date and pick the latest file.
